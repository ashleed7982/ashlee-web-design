Art Tips and Stuff
    Some art tips: DON'T TRACE. #1 RULE.
  • Reference whatever you want to draw!
  • Practice any time you can.
  • Draw what YOU want to draw, not what others say to draw.
  • If you try shading, use light references online.
  • Don't be afraid to ask for help from others! In real life or online.
